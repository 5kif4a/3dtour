Политика конфиденциальности

Функции приложения
‍
‍После авторизации в приложении пользователю предоставляется доступ к следующим функциям:
- Сбор данных о качестве сотовой связи (2G, 3G, 4G, 5G) и определение     местоположения.
- Отправка собранных данных на сервер для анализа и отображения на интерактивной карте.
‍
‍Типы собираемых данных
‍
‍Мы собираем следующие данные:
- Данные о местоположении: Географические координаты для определения зоны покрытия сети.
- Информация о сети: Данные о качестве сигнала (уровень, тип связи).
- Информация о состоянии устройства: Данные о текущем состоянии сети     устройства (например, доступные сети, скорость передачи данных).

‍Цели использования данных
‍
‍Собранные данные используются исключительно для: Обеспечения     корректной работы приложения. Анализа и улучшения качества связи. Визуализации информации на интерактивной карте. Мы не передаем данные третьим лицам, за исключением случаев, предусмотренных законом.


‍Хранение и передача данных

Хранение данных
‍
- Данные хранятся локально на вашем устройстве до момента их отправки на сервер.
- После отправки данные удаляются из локального хранилища устройства.
‍
Передача данных
‍
- Данные передаются на сервер с использованием зашифрованного соединения.
- На сервере данные хранятся на защищенных системах и используются исключительно для анализа и улучшения работы приложения.

Мы гарантируем, что данные не передаются третьим лицам без вашего согласия.


‍Запрашиваемые разрешения
‍
‍Для обеспечения работы приложения требуется предоставить следующие разрешения:

‍Доступ к местоположению:

ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION, ACCESS_BACKGROUND_LOCATION — для определения точного местоположения, включая сбор данных в фоновом режиме.  
‍
‍Доступ к сети:  

INTERNET, ACCESS_NETWORK_STATE — для передачи данных на сервер и анализа сетевого состояния.  
‍
‍Доступ к состоянию телефона:  

READ_PHONE_STATE — для анализа состояния сети и качества связи.  
‍
‍Работа в фоновом режиме:  

FOREGROUND_SERVICE, FOREGROUND_SERVICE_LOCATION — для сбора данных в фоновом режиме. Разрешения используются исключительно для обеспечения работы приложения и выполнения его функциональности.


‍Защита данных
‍
‍Мы принимаем все возможные меры для защиты ваших данных:
‍
‍Шифрование передачи данных: 

Все данные передаются на сервер через защищенные каналы с использованием протоколов шифрования (например, HTTPS).

‍Защита серверов: 

Данные хранятся на защищенных серверах, которые соответствуют современным стандартам безопасности и соответствуют требованиям TIER 3.

‍Ограничение доступа: 

Доступ к данным имеют только уполномоченные сотрудники с целью их обработки.
